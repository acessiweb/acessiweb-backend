This is a REST API for my final project in the Information Systems course.

## Project Setup

```sh
npm install
```

### Start Server for Development

```sh
npm run start:dev
```

### Compile for Production

```sh
npm run start:prod
```

### Run unit tests

```sh
npm run test:unit
```

### Run integration tests

```sh
npm run test:integration
```
